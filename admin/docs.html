API Documentation

Welcome to the WhatsApp Gateway API. This simplified API allows you to send messages and manage groups using simple Query Parameters or JSON payloads.

Ensure your session is Connected in the Dashboard before making API calls.

Send Message API

The primary endpoint for sending WhatsApp messages. Supports GET and POST methods.

GET/POST /api/v1/send-message

Base URL: http://your-domain.com/api/v1/send-message

Parameters
Parameter Type Description
apikey String *Required. Your session API Token.
mtype Enum *Required. Type of message. Options:
text, image, video, audio,
document, sticker
receiver String *Required. Receiver Phone Number (e.g. 62812...) or Group ID (e.g. ...g.us).
text String Required if mtype is text. The content of your message.
url Url Required for media types. Direct link to file (e.g. http://example.com/image.jpg).
filename String Optional. Custom filename for document messages.
Example Usage (n8n / cURL)

Recommended format for n8n HTTP Request nodes (Body Content Type: Form-URL-Encoded).

1. Send Text Message
curl -X POST "http://localhost:3000/api/v1/send-message" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "apikey=YOUR_API_KEY" \
  -d "mtype=text" \
  -d "receiver=628123456789" \
  -d "text=Hello World From API"
2. Send Image Media
curl -X POST "http://localhost:3000/api/v1/send-message" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "apikey=YOUR_API_KEY" \
  -d "mtype=image" \
  -d "receiver=628123456789" \
  -d "url=https://example.com/image.jpg" \
  -d "text=Optional Caption Here"

Group Management API

Endpoints to retrieve group information.

GET/POST /api/v1/fetch-group

Get a list of all groups joined by the session.

Parameter Description
apikey *Required. Your session API Token.
GET/POST /api/v1/fetch-member

Get list of members in a specific group.

Parameter Description
apikey *Required. Your session API Token.
idgroup *Required. The Group ID (e.g. 12345@g.us).
